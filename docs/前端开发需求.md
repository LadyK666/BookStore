## 网上书店管理系统 - 前端开发需求

> 本文档记录前端界面的功能需求，与后端 JDBC 数据访问层对接。

### 一、总体架构

根据课程要求：
- **管理员端（C/S 模式）**：书店内部日常处理业务，使用 JavaFX 桌面应用实现。
- **顾客端（B/S 模式）**：顾客通过网上进行订书及查询浏览，可使用 JavaFX 桌面应用模拟（或后续扩展为 Web）。

### 二、用户类型

系统支持两种用户类型：

1. **管理员（Admin）**
   - 登录后进入管理后台
   - 可进行库存、采购、客户、订单、发货、供应商等管理操作

2. **顾客（Customer）**
   - 登录后进入购物界面
   - 可进行书目浏览、查询、下单、查看订单等操作

### 三、顾客端功能需求

#### 3.1 登录/注册
- 顾客使用用户名和密码登录
- 新顾客可以注册账号

#### 3.2 书目浏览与查询
- 显示所有在售书目列表（书名、作者、出版社、价格等）
- 支持按书名、作者、出版社、关键字进行搜索
- 点击书目可查看详情

#### 3.3 购物车与下单
- 选择书目并指定数量，加入购物车
- 查看购物车，可修改数量或删除
- 提交订单：
  - 显示应付金额（根据信用等级折扣计算）
  - 确认收货地址
  - 确认下单

#### 3.4 订单管理
- 查看我的订单列表
- 查看订单详情（订单状态、明细、发货信息）
- 待付款订单可进行付款操作

#### 3.5 个人中心
- 查看/修改个人信息
- 查看账户余额和信用等级
- 查看消费记录

### 四、管理员端功能需求

#### 4.1 登录
- 管理员使用专用账号登录

#### 4.2 书目与库存管理
- 查看所有书目列表
- 新增书目（含作者、关键字）
- 编辑书目信息
- 查看库存情况
- 调整库存数量

#### 4.3 采购管理
- 查看缺书记录列表
- 手动添加缺书记录
- 根据缺书记录创建采购单
- 处理采购到货（更新库存）

#### 4.4 客户管理
- 查看所有客户列表
- 查看客户详情（余额、信用等级、订单历史）
- 调整客户信用等级
- 为客户充值（增加账户余额）

#### 4.5 订单管理
- 查看所有订单列表
- 按状态筛选订单（待付款、待发货、已发货、已完成）
- 查看订单详情

#### 4.6 发货管理
- 查看待发货订单
- 执行发货操作（填写物流信息）
- 查看发货记录

#### 4.7 供应商管理
- 查看供应商列表
- 新增/编辑供应商信息
- 管理供货关系（供应商可供应哪些书目）

### 五、界面设计原则

1. **简洁实用**：界面简洁，功能直观
2. **统一风格**：使用一致的配色和布局
3. **响应反馈**：操作后给予明确的成功/失败提示
4. **数据验证**：输入数据进行基本校验

### 六、技术选型

- **GUI 框架**：JavaFX（JDK 21 内置支持，或使用 OpenJFX）
- **数据访问**：复用现有 DAO 层
- **布局**：使用 FXML 或纯 Java 代码构建界面
